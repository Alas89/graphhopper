GraphHopper - Java graph routing engine
GraphHopper offers memory efficient algorithms and data structures suited for road routing. It stands under the Apache License and is heavily unit tested although still in alpha.
The Core
A small library ready to be thrown into your Java application. Be it a server-side, desktop or Android one.
  • Easy to use and small
  • Memory efficient
  • Well tested
  • .. and fast. In that order.
From Web
Clone this example on how to use GraphHopper within a web-based application using Leaflet.
Or use any API which supports GeoJSON or create your own small adapter-servlet.
Areas of 1000km2 are possible and with your help this will increase.
For Android
Offline road routing example for Android using Mapsforge.
Clone this ready-for-Eclipse example to run on Android.
With memory mapped files it is possible to serve the graph in a heap friendly but fast manner.
Areas of 150km2 are possible.
