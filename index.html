GraphHopper: a routing engine for roads and more

GraphHopper offers memory efficient algorithms in Java for road routing e.g. Dijkstra and A* but also optimized shortest path algorithms like Contraction Hierarchies. It stands under the Apache License and is heavily unit tested although still in alpha - ie. the library itself is quite stable but the API not.
The Core
A small library ready to be thrown into your Java application. Be it a server-side, desktop or Android one.
  • Easy to use and small
  • Memory efficient
  • Well tested
  • .. and fast. In that order.
