GraphHopper - Java graph routing engine
GraphHopper offers memory efficient algorithms and data structures suited for road routing. It stands under the Apache License and is heavily unit tested although still in alpha (ie. API isn't stable yet).
The Core
A small library ready to be thrown into your Java application. Be it a server-side, desktop or Android one.
  • Easy to use and small
  • Memory efficient
  • Well tested
  • .. and fast. In that order.
For the Web
Clone this example on how to use GraphHopper within a web-based application using Leaflet. Using alternatives like OpenLayers or Google is easy too.

German-sized road networks have been already tried without any problems.
For Android
Offline road routing example for Android using Mapsforge.
Clone this Eclipse project to get started on Android.

With memory mapped files it is possible to serve the graph in a heap friendly but fast manner.
